# Book Tracker - Трекер прочитанных книг

## Описание программы
Book Tracker - это графическое приложение для управления списком прочитанных книг. Программа позволяет:
- Добавлять книги с указанием названия, автора, жанра и количества страниц
- Редактировать и удалять записи о книгах
- Фильтровать книги по жанру и количеству страниц
- Сохранять данные в JSON файл для последующего использования
- Просматривать детальную информацию о каждой книге

## Технологии
- Python 3.x
- Tkinter (GUI)
- JSON (хранение данных)
- Git (контроль версий)
